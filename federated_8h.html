typedef lf_ret_t(* deserialize_hook) (void *user_struct, const unsigned char *msg_buffer, size_t msg_size)

A function type for deserializers that takes a message buffer and deserializes them to a port value.

The default implementation is a raw memcpy, but this can be overridden with a custom implementation, or with protobufs.

Parameters
user_structA pointer to the port value into which the message should be deserialized
msg_bufferA pointer to the buffer with the serialized message
msg_sizeThe size of the serialized message
Returns
Whether the deserialization was successful

typedef size_t(* serialize_hook) (const void *user_struct, size_t user_struct_size, unsigned char *msg_buffer)

A function type for serializers that takes port values and serializes them into a message buffer.

The default implementation is a raw memcpy, but this can be overridden with a custom implementation, or with protobufs.

Parameters
user_structA pointer to the port value to serialize
user_struct_sizeThe size of the port value to serialize
msg_bufferA pointer to the buffer to write the serialized message to
Returns
The size of the serialized message
